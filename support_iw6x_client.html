IW6x CLIENT INSTALL GUIDE

If You Do Not Own The Game On Steam

Getting Started:

  1. Download and Install qBittorrent.
  2. Open the full game torrent in qBittorrent, making sure to match these settings.
  3. Once your torrent says Seeding, click on the torrent and hit the pause button.
  4. Download the X Labs Launcher.
  5. Select the Settings Cog, then click Browse on Ghosts Installation.
  6. Select your game folder.
  7. To start the game, Select the IW6x Icon, then press either Singleplayer or Multiplayer.

To change your name, open the in-game console (press the ~ key) and type name yourname.

To unlock all, open the in-game console (press the ~ key) and type unlockstats.

Custom classes are made in the Private Match section.
